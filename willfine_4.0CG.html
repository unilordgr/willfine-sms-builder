Willfine 4.0CG (older firmware) — SMS Command Builder

Tick the options you want, then copy the generated SMS command. Different colour scheme + extra "Send to Phone" tab to distinguish from the newer T4.0CG tool.

Send to Phone ($06)

Basic command prefix: $01*27# — 27 parameters, separated by #, terminated by $.
GPRS command prefix: $10*13# — 13 parameters.
Send-to-Phone command prefix: $06*8# — 4 phone numbers + 4 reserved fields.
Only takes effect when MMS is set to ON in the GPRS tab.
Send-to-Email command prefix: $08*8# — 4 email addresses + 4 reserved fields.
Fixed trigger command. No options to set.
1 Get a Picture
Trigger the camera to take a photo / video right now. Sent on its own — not part of any longer string.

Decode a command

Paste a command string (e.g. $01*27#1#3#2#5#1#2#0#1#0#1#0#0#0#0#OFF#OFF#OFF#OFF#1#0#1#OFF#OFF#0#0#0#0$) to see what it tells the camera to do.
